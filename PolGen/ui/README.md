# Папка, в которой хранятся UI-компоненты интерфейса.

### В этой папке содержатся все необходимые файлы для функционирования основного WebUI-интерфейса, созданного с использованием библиотеки Gradio.

### На данный момент доступен только WebUI-интерфейс, но в будущем планируется также реализация графического интерфейса (GUI).
